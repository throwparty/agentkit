# 👋 Welcome

AgentKit provides a set of composable tools for building your own agentic software engineering environment, or for plugging in to your preferred environment. Each tool implements the relevant standards to make it reasonably interoperable.

- [Lens](lens): provides `fetch` and `search` tools backed by various search engines.
- [Litterbox](litterbox): review *outputs*, not *actions*: give your AI agents litter trays to poop into.
